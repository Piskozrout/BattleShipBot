# BattleShipBot
Battleship game bot :ship:

:warning: The most unoptimized code you will ever see. :warning:

Programmed for a coding competition, but I didn't manage to finish it. The basic loop is functional.

It was designed to receive a string of 144 characters representing tiles on map (water, ship, blank) and send coordinates for the shot. I added visualization of probability (the numbers) and map (the "*").
